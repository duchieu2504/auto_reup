# Kế hoạch triển khai Frontend: Modern Dashboard

Mục tiêu: Xây dựng giao diện bảng điều khiển (Dashboard) quản trị hệ thống Auto Re-up theo phong cách UI/UX hiện đại, tối ưu trải nghiệm người dùng (Smooth Animations, Glassmorphism).

## 1. Nền tảng và Công cụ
- **Framework:** Vite + ReactJS.
- **Styling:** Vanilla CSS (để tùy chỉnh cao nhất).
- **Màu sắc:** Dark Mode (Đen/Xám đậm) kết hợp tông Cyberpunk (Tím/Xanh Neon).

## 2. Cấu trúc Source Code
- Khởi tạo React project mới trong folder `frontend/`.
- Cài đặt `lucide-react` cho icons và `react-router-dom` cho routing.
- Cấu hình font chữ **Inter**.

## 3. Các trang chính
- **Tổng quan (Overview):** Hiển thị thống kê.
- **Phase 1 (Cào dữ liệu):** Form nhập link và thao tác.
- **Cấu hình (Settings):** Cài đặt Cookie/Proxy.
